# TesloDB - NestJS

1. Clone the repo

2. ```yarn install```

3. rename ```..env.teamplate```. to ```.env ```and fill the variables

4. Up the docker-compose ``` docker-compose up -d ```

5. dev mode ```yarn start:dev```
